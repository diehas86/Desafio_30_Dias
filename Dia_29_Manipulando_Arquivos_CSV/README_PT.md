O que este código faz
Este programa processa uma lista de registros de acesso de usuários, extraindo nome, tempo e ação realizada (login ou logout). Os dados são organizados em uma estrutura de dicionário e, ao final, é gerado um relatório resumido por usuário, contendo tempo total, quantidade de logins, quantidade de logouts e status final de atividade.

Objetivo
Este exercício demonstra a capacidade de manipular dados textuais estruturados, organizar informações utilizando dicionários e gerar relatórios consolidados com base em valores acumulados.

Conceitos praticados

laços de repetição (for)

separação de strings

criação e atualização de dicionários

lógica condicional (if/else)

manipulação de listas

agregação de dados

Observações para recrutadores
Exercício do Dia 29 com foco na consolidação dos fundamentos da linguagem Python. A intenção é demonstrar organização lógica, clareza estrutural e domínio consistente de conceitos essenciais esperados de um desenvolvedor júnior.

What this code does

This code processes a list of users and validates basic fields such as name, age, email, and active status. Each user is classified as valid or invalid based on predefined rules, and a summary of the results is displayed.

Purpose

The exercise demonstrates how to iterate over structured data and apply multiple validation rules in a clear and organized way, simulating a common data validation task.

Concepts practiced

for loops

dictionaries and lists

boolean variables

conditional statements

basic data validation

Notes for reviewers

This is a study-level exercise focused on logic clarity and readability. The goal is to demonstrate foundational Python skills expected from a junior-level candidate.

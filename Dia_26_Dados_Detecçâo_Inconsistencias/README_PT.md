O que este código faz

Este código percorre uma lista de usuários e valida campos básicos como nome, idade, email e status ativo. Cada usuário é classificado como válido ou inválido de acordo com regras definidas, e um resumo final é exibido.

Objetivo

O exercício demonstra como iterar sobre dados estruturados e aplicar múltiplas regras de validação de forma clara e organizada, simulando um cenário comum de validação de dados.

Conceitos praticados

laço for

dicionários e listas

variáveis booleanas

estruturas condicionais

validação básica de dados

Observações para recrutadores

Este é um exercício de estudo com foco em lógica e legibilidade. A intenção é demonstrar fundamentos sólidos de Python esperados de um desenvolvedor júnior.

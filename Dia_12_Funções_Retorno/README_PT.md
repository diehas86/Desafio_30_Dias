O que este código faz
Implementa um sistema simples em console para registrar notas de um aluno, calcular a média e determinar se o aluno foi aprovado ou reprovado com base nessa média.

Objetivo
Demonstrar o uso correto de funções com retorno, a passagem de dados entre funções e o controle básico do fluxo do programa por meio de um menu.

Conceitos praticados

Funções com parâmetros e retorno

Uso de listas para armazenar dados

Estruturas condicionais

Validação básica de entrada

Controle de fluxo com laço while e menu

Observações para recrutadores
Este é um exercício de aprendizado com foco na consolidação dos fundamentos de Python. A ênfase está na clareza do código, na lógica correta e na compreensão de como funções se comunicam por meio de retornos.

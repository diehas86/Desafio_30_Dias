O que este código faz
Processa uma lista de registros de vendas, agrupa os dados por produto, calcula a quantidade total e o faturamento por produto, o faturamento geral e identifica os produtos com maior e menor faturamento.

Objetivo
Demonstrar manipulação de dados estruturados, lógica de agregação e geração de um relatório simples em nível Python Júnior.

Conceitos praticados

Listas e dicionários

Laços de repetição e condicionais

Agregação e acumulação de dados

Cálculos numéricos básicos

Normalização de strings

Observações para recrutadores
Exercício de aprendizado com foco em base sólida de Python. O código é intencionalmente simples, priorizando clareza e correção, como parte de uma evolução diária voltada a entrevistas de nível júnior.

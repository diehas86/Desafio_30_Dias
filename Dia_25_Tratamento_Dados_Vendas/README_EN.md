What this code does
Processes a list of sales records, groups data by product, calculates total quantity and revenue per product, computes overall revenue, and identifies the products with the highest and lowest revenue.

Purpose
Demonstrates handling structured data, aggregation logic, and basic reporting in a Python Junior context.

Concepts practiced

Lists and dictionaries

Loops and conditional logic

Data aggregation and accumulation

Basic numeric calculations

String normalization

Notes for reviewers
This is a learning exercise focused on building a solid Python foundation. The code prioritizes clarity and correctness over optimization and reflects daily practice aimed at junior-level interviews.

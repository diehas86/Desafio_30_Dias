What this code does
Collects user information (name, age, height, employment status) and displays each value’s data type immediately after input. Provides a summary of all inputs for review.

Purpose
Demonstrates understanding of basic Python data types and input handling, including strings, integers, floats, and boolean logic representation.

Concepts practiced

Input and output

Type conversion (int, float)

String manipulation (strip)

Boolean evaluation

Formatted printing (f-strings)

Notes for reviewers
Python Junior level exercise focused on data types, clear input/output handling, and preparing for structured coding assessments.

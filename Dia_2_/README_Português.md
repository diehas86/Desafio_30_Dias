O que este código faz

Coleta informações do usuário (nome, idade, altura, situação de emprego) e exibe o tipo de cada valor logo após a entrada. Fornece um resumo de todos os dados para revisão.

Objetivo
Demonstra compreensão dos tipos básicos de dados em Python e manipulação de entrada, incluindo strings, inteiros, floats e representação lógica booleana.

Conceitos praticados

Entrada e saída de dados

Conversão de tipos (int, float)

Manipulação de strings (strip)

Avaliação booleana

Impressão formatada (f-strings)

Observações para recrutadores
Exercício de nível Python Júnior, focado em tipos de dados, entrada/saída clara e preparação para avaliações estruturadas.

O que este código faz

Processa uma lista de registros de acesso, conta quantas vezes cada usuário acessou o sistema e separa tentativas de acesso bem-sucedidas e falhas.

Objetivo

Demonstrar lógica básica de processamento de logs, com foco em extração de dados, contagem de ocorrências e uso de condicionais, como é comum em testes técnicos para Python Júnior.

Conceitos praticados

Separação de strings

Listas e dicionários

Contagem de ocorrências

Estruturas condicionais

Laços de repetição

Observações para recrutadores

Exercício de nível júnior voltado ao fortalecimento de fundamentos de Python. A ênfase está na legibilidade, na lógica aplicada e na capacidade de manipular dados simples de forma correta.

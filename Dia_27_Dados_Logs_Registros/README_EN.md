What this code does

Processes a list of access log records, counts how many times each user accessed the system, and separates successful and failed access attempts.

Purpose

Demonstrate basic log processing logic, focusing on data extraction, counting occurrences, and simple conditional handling, as commonly required in junior-level Python interview tasks.

Concepts practiced

String splitting

Lists and dictionaries

Counting occurrences

Conditional logic

Looping over collections

Notes for reviewers

This is a junior-level exercise aimed at reinforcing fundamental Python logic and data handling. The focus is on clarity, correctness, and reasoning rather than optimization or advanced features.

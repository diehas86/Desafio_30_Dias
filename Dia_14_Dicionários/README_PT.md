O que este código faz
Implementa um cadastro simples em linha de comando que permite adicionar usuários com nome e idade, buscar um usuário específico, listar todos os usuários cadastrados e encerrar o programa.

Objetivo
Demonstrar o uso básico de dicionários para armazenar e acessar dados estruturados informados pelo usuário por meio de um menu.

Conceitos praticados

Dicionários (estrutura chave-valor)

Entrada e saída de dados

Conversão de tipos

Estruturas condicionais

Controle de laços

Tratamento básico de erros com try/except

Observações para recrutadores
Este exercício é de nível introdutório e tem como foco o entendimento prático de dicionários e fluxo de controle em Python. A prioridade é a clareza do código e o domínio dos fundamentos.

O que este código faz

Este código implementa um menu simples em linha de comando que permite ao usuário gerenciar uma lista de compras.
O usuário pode adicionar itens, remover itens, verificar se um item existe na lista, mostrar todos os itens ou encerrar o programa.

Objetivo

Este exercício demonstra o uso básico de listas em conjunto com laços de repetição e condicionais para criar um programa interativo baseado em menu, com foco em lógica fundamental e clareza do código.

Conceitos praticados

Listas

Laço while

Estruturas condicionais if / elif / else

Entrada de dados pelo usuário

Operações básicas com listas (append, remove, verificação com in)

Observações para recrutadores

Este código foi desenvolvido como parte de um processo de aprendizado em Python para nível Júnior, com foco em preparação para entrevistas técnicas.
A prioridade foi a construção da lógica e a legibilidade do código, sem o uso de validações avançadas ou otimizações.
Tratamento de exceções, normalização de entradas, ordenação alfabética e controle de itens duplicados não foram implementados de forma intencional, pois não faziam parte do escopo do exercício neste nível.

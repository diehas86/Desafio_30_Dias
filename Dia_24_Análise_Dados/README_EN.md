What this code does

Analyzes a fixed list of daily sales values and produces a summary including total sales, average sales, highest sale, lowest sale, and the number of days with sales above the average.

Purpose

Demonstrates basic data analysis logic using Python fundamentals without relying on built-in aggregation functions.

Concepts practiced

Lists

For loops

Conditional statements

Accumulators

Basic arithmetic operations

Output formatting with f-strings

Notes for reviewers

This is a beginner-level exercise focused on clarity, readability, and correct logical reasoning, designed as part of preparation for Python Junior technical interviews.

O que este código faz

Analisa uma lista fixa de valores de vendas diárias e gera um resumo contendo o total vendido, a média de vendas, a maior venda, a menor venda e a quantidade de dias com vendas acima da média.

Objetivo

Demonstrar lógica básica de análise de dados utilizando fundamentos de Python, sem o uso de funções prontas de agregação.

Conceitos praticados

Listas

Laços for

Estruturas condicionais

Variáveis acumuladoras

Operações matemáticas básicas

Formatação de saída com f-strings

Observações para recrutadores

Exercício de nível iniciante com foco em clareza, legibilidade e raciocínio lógico, desenvolvido como parte da preparação para entrevistas técnicas de Python Júnior.

O que este código faz

Este script recebe um texto do usuário, garante que a entrada não esteja vazia, remove pontuações básicas e analisa as palavras para identificar a quantidade total de palavras, além da maior e da menor palavra do texto.

Objetivo

Este exercício demonstra fundamentos de manipulação de strings e lógica básica de análise de dados, esperados de um candidato a Python Júnior.

Conceitos praticados

Validação de entrada do usuário

Limpeza de strings com translate

Separação de strings em listas

Laço for

Comparações condicionais

Raciocínio lógico básico

Observações para recrutadores

Código desenvolvido como parte de prática diária em Python, com foco em legibilidade, organização e preparo para entrevistas técnicas de nível Júnior.

What this code does

This script reads a text input from the user, ensures the input is not empty, removes basic punctuation, and analyzes the words to determine the total number of words, as well as the longest and shortest word in the text.

Purpose

This exercise demonstrates fundamental string processing and basic data analysis logic commonly expected from a Python Junior candidate.

Concepts practiced

User input validation

String cleaning with translate

Splitting strings into lists

Looping with for

Conditional comparisons

Basic algorithmic reasoning

Notes for reviewers

This code was written as part of daily Python practice, with a focus on clarity, correctness, and interview-ready logic rather than advanced optimizations or libraries.

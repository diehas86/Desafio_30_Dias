O que este código faz

Implementa um menu simples em terminal que permite ao usuário escolher entre opções pré-definidas, valida a entrada numérica, executa ações conforme a escolha e mantém o sistema em execução até que o usuário decida sair.

Objetivo

Este exercício demonstra como estruturar um pequeno sistema interativo utilizando fluxo de controle claro, validação de entradas e separação básica de responsabilidades, seguindo boas práticas de Python Júnior.

Conceitos praticados

Validação de entrada do usuário

Controle de laço while

Estruturas condicionais (if / elif / else)

Uso de funções com responsabilidade definida

Interação básica via terminal

Observações para recrutadores

Trata-se de um exercício com foco em aprendizado, desenvolvido em nível Python Júnior, priorizando clareza, organização e lógica correta em vez de complexidade de sistema.

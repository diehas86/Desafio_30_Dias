O que este código faz

Este programa solicita ao usuário seu nome, idade e profissão, armazena essas informações em variáveis e exibe uma mensagem formatada com os dados informados.

Objetivo

Este exercício demonstra o uso correto de entrada e saída de dados em Python, com foco na clareza e no entendimento dos conceitos fundamentais.

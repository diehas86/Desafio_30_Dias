O que este código faz

Este código solicita ao usuário a descrição de tarefas concluídas e imprime cada uma durante a execução, simulando uma lista simples de tarefas.

Objetivo

Este exercício demonstra como percorrer valores pré-definidos e coletar entradas do usuário quando o uso de listas é intencionalmente restrito.

Conceitos praticados

Laço for

Iteração com tupla

Entrada de dados com input()

Formatação de strings com f-strings

Atribuição básica de variáveis

Observações para recrutadores

Este código foi desenvolvido propositalmente sem o uso de listas.
A tupla foi utilizada apenas como uma sequência fixa para controlar o laço e representar visualmente a ordem das tarefas.
Cada descrição é exibida imediatamente, pois o armazenamento de múltiplos valores estava fora das regras do exercício.
O foco é aprendizado sob restrições, não uma solução final de produção.

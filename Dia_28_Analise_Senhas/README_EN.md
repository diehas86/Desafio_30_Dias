What this code does
Validates a list of user passwords based on two criteria: minimum length of 8 characters and presence of at least one numeric digit. It classifies each password as valid or invalid and displays a summary of the results.

Purpose
Demonstrates the ability to implement conditional validation logic using loops and control flow structures without relying on external libraries.

Concepts practiced

for loops

Nested loops

Conditional statements (if/else)

Boolean flag variables

String methods (.isdigit())

List manipulation with .append()

Basic data aggregation using len()

Notes for reviewers
This exercise focuses on fundamental logical structuring and validation flow expected from a Junior Python candidate. The goal is to demonstrate clear reasoning, control flow organization, and clean implementation of basic requirements.

O que este código faz
Valida uma lista de senhas com base em dois critérios: mínimo de 8 caracteres e presença de pelo menos um número. Classifica cada senha como válida ou inválida e exibe um resumo dos resultados.

Objetivo
Demonstrar a implementação de lógica de validação utilizando estruturas de repetição e condicionais, sem uso de bibliotecas externas.

Conceitos praticados

Laço for

Laço aninhado

Estruturas condicionais (if/else)

Variável booleana de controle

Método de string (.isdigit())

Manipulação de listas com .append()

Uso de len() para agregação simples de dados

Observações para recrutadores
Este exercício foca na construção lógica fundamental esperada de um candidato Python Júnior. A intenção é demonstrar organização de fluxo, clareza na validação de regras e implementação direta dos requisitos.

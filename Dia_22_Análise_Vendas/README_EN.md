What this code does

This program records daily sales values entered by the user and generates a simple summary report with totals, averages, and comparisons.

Purpose

This exercise demonstrates the ability to collect user input, process numerical data, and produce a clear report using fundamental Python logic.

Concepts practiced

While loops for continuous input

Lists for data storage

Conditional logic

Basic numeric calculations (total, average, min, max)

String handling for user input

Notes for reviewers

This is a beginner-level Python exercise focused on clarity, logical flow, and correct use of core language features, designed as part of preparation for Python Junior technical interviews.

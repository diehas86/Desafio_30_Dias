O que este código faz

Este programa registra valores de vendas diárias informados pelo usuário e gera um relatório simples com totais, média e comparações entre os valores.

Objetivo

Este exercício demonstra a capacidade de coletar dados do usuário, processar valores numéricos e apresentar um relatório claro utilizando lógica fundamental em Python.

Conceitos praticados

Laço while para entrada contínua de dados

Listas para armazenamento

Estruturas condicionais

Cálculos numéricos básicos (total, média, maior e menor valor)

Manipulação de strings para entrada do usuário

Observações para recrutadores

Trata-se de um exercício de nível iniciante, com foco em organização, clareza de raciocínio e uso correto dos fundamentos da linguagem, voltado à preparação para entrevistas de Python Júnior.

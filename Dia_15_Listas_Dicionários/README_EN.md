What this code does
Provides a simple command-line system to register customer orders and display a list of saved orders during program execution.

Purpose
Demonstrates the creation of a basic interactive menu, data validation, and storage of structured data using Python fundamentals.

Concepts practiced

while loop

Conditional statements (if/elif/else)

Lists and dictionaries

User input handling

Type conversion and error handling (try/except)

Notes for reviewers
This is a beginner-level Python exercise focused on building a clear and functional menu-driven program, reinforcing core logic and data organization skills expected at a Python Junior level.

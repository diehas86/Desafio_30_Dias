O que este código faz
Fornece um sistema simples em linha de comando para cadastrar pedidos de clientes e exibir a lista de pedidos armazenados durante a execução do programa.

Objetivo
Demonstrar a criação de um menu interativo básico, validação de dados e armazenamento de informações estruturadas usando fundamentos de Python.

Conceitos praticados

Laço while

Estruturas condicionais (if/elif/else)

Listas e dicionários

Entrada de dados do usuário

Conversão de tipos e tratamento de erros (try/except)

Observações para recrutadores
Este é um exercício de nível iniciante em Python, com foco em lógica clara, organização de dados e construção de programas interativos, alinhado às expectativas de um Python Júnior.

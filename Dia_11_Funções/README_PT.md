O que este código faz
Este código implementa um sistema simples em terminal para cadastrar notas de alunos, calcular a média das notas e exibir o resultado final (aprovado ou reprovado) com base nessa média.

Objetivo
Este exercício demonstra como organizar um programa simples utilizando funções, melhorando a legibilidade, a reutilização da lógica e o controle do fluxo do programa.

Conceitos praticados

Definição e chamada de funções

Uso de listas para armazenamento de dados

Estruturas condicionais

Laços de repetição para controle de menu

Retorno de valores em funções

Observações para recrutadores
Trata-se de um exercício de nível iniciante, voltado à construção de uma base sólida em Python para entrevistas de nível Júnior, com foco em organização de código e responsabilidade clara das funções.

What this code does
This code implements a simple console-based system to register student grades, calculate their average, and display the final result (approved or not approved) based on the calculated average.

Purpose
This exercise demonstrates how to organize a small program using functions, improving readability, reuse of logic, and control of program flow.

Concepts practiced

Function definition and function calls

Lists for data storage

Conditional statements

Loops for menu control

Returning values from functions

Notes for reviewers
This is a beginner-level Python exercise focused on building a solid foundation for junior-level interviews, emphasizing code organization and clear responsibilities for each function.

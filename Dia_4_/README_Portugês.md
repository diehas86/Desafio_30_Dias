O que este código faz:

Verifica se um usuário pode acessar um programa com base na idade, na existência de cadastro e na aceitação dos termos de uso. Caso algum requisito não seja atendido, o acesso é negado; caso contrário, é liberado.

Objetivo:

Demonstrar lógica básica de decisão utilizando entrada de dados e estruturas condicionais em Python.

Conceitos praticados

Entrada de dados pelo usuário

Conversão de tipos (string para inteiro)

Operadores lógicos (or)

Estruturas condicionais (if / else)

Observações para recrutadores:
Este é um exercício simples, focado nos fundamentos do Python. O tratamento avançado de entradas foi mantido fora do escopo de forma intencional, para respeitar o enunciado e evitar complexidade desnecessária. O objetivo é evidenciar raciocínio lógico claro em nível Python Júnior.

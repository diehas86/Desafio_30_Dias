O que este código faz
Implementa um sistema de gerenciamento de usuários via terminal, permitindo criar, listar, buscar, atualizar e remover usuários, com validações de entrada e armazenamento estruturado.

Objetivo
Demonstra o tratamento de entrada de usuário, aplicação de regras de validação, organização do código em funções e gerenciamento de dados com listas e dicionários.

Conceitos praticados

Funções com responsabilidades claras

Listas e dicionários

Validações de entrada

Laços (while, for)

Operações básicas de CRUD

Observações para recrutadores
Código reflete um iniciante em Python evoluindo para nível Júnior sólido, com foco em clareza, organização e tratamento de erros.

O que este código faz
Implementa um gerenciador simples de tarefas em linha de comando, permitindo adicionar, listar e remover tarefas por meio de um menu numérico.

Objetivo
Demonstrar a construção de um programa interativo com execução contínua, validação de entrada e manipulação básica de dados.

Conceitos praticados

Laço while

Listas

Estruturas condicionais

Validação de entrada

enumerate

Interação básica via terminal

Observações para recrutadores
Este é um exercício de nível iniciante em Python, com foco em consolidar fundamentos da linguagem e controle de fluxo claro. A intenção é mostrar base sólida e evolução gradual.

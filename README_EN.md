30-Day Challenge – Python

This repository contains 30 daily Python challenges designed to practice logic and core language fundamentals.

Each folder represents one day of study and includes:

A Python script

A README in Portuguese and English

The exercises focus on reasoning. In some challenges, the use of libraries was intentionally avoided to strengthen logical foundations.

What this repository demonstrates

Understanding of Python fundamentals

Consistent use of conditionals and loops

Code organization and readability

Progressive improvement over time

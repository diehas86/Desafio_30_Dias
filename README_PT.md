Desafio 30 Dias – Python

Este repositório reúne 30 desafios diários em Python, criados para praticar lógica e fundamentos da linguagem.

Cada pasta representa um dia de estudo e contém:

Um script em Python

Um README em Português e Inglês

Os exercícios foram feitos com foco em raciocínio. Em alguns desafios, o uso de bibliotecas foi propositalmente evitado para reforçar a base lógica.

O que este repositório demonstra

Compreensão dos fundamentos de Python

Uso consistente de condicionais e laços

Organização e legibilidade de código

Evolução progressiva ao longo dos dias

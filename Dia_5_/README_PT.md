O que este código faz
Solicita a idade do usuário e determina se o acesso é permitido com base em uma idade mínima exigida.

Objetivo
Demonstrar lógica básica de decisão em Python utilizando uma regra condicional simples e clara.

Conceitos praticados

Entrada de dados

Conversão de tipos

Estruturas condicionais (if / else)

Operadores de comparação

Observações para recrutadores
Este é um exercício de nível iniciante em Python, focado em fundamentos. O objetivo é clareza, lógica correta e capacidade de justificar decisões em entrevistas para vagas júnior.

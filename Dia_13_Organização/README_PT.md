O que este código faz
Este código implementa um gerenciador simples de tarefas em linha de comando. Ele permite adicionar tarefas, listar todas as tarefas cadastradas com numeração e encerrar o programa por meio de um menu interativo. O sistema também trata entradas inválidas, como opções não numéricas e descrições de tarefas vazias.

Objetivo
Este exercício demonstra como estruturar um pequeno programa interativo em Python, utilizando funções bem definidas, validações básicas de entrada e um fluxo de controle legível, dentro do esperado para um Python Júnior.

Conceitos praticados

Funções e separação de responsabilidades

Laço while para execução contínua

Estruturas if / elif / else

Listas e operações básicas com listas

Validação de entrada com try / except (ValueError)

Avaliação booleana de valores vazios

Uso de enumerate para exibição com índice

Observações para recrutadores
Este código faz parte de uma prática diária voltada à construção de uma base sólida em Python para nível Júnior. O foco está na clareza, correção e capacidade de explicar decisões técnicas, e não no uso de recursos avançados.

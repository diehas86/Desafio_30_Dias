O que este código faz

Implementa um menu simples em console que permite cadastrar nomes em uma lista, listar todos os nomes cadastrados ou encerrar o programa.

Objetivo

Demonstrar refatoração básica, organizando um código funcional em funções mais claras, melhorando a legibilidade sem alterar o comportamento do programa.

Conceitos praticados

Laço while para controle de menu

Estruturas condicionais

Listas

Funções

Validação básica de entrada

Observações para recrutadores

Este é um exercício inicial de refatoração, com foco em organização e clareza do código. O objetivo é mostrar fundamentos esperados de um candidato a Python Júnior.

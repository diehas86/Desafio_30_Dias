What this code does

Implements a simple console-based menu that allows the user to register names in a list, display all registered names, or exit the program.

Purpose

Demonstrate basic refactoring by organizing a functional but simple program into clearer functions, improving readability without changing its behavior.

Concepts practiced

While loop for menu control

Conditional statements

Lists

Functions

Basic input validation

Notes for reviewers

This is an early-level refactoring exercise focused on code organization and clarity. The goal is not complexity, but demonstrating understanding of structure expected from a Python Junior candidate.

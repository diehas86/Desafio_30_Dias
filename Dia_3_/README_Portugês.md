O que este código faz

Recebe dois valores numéricos do usuário, converte a entrada para tipos numéricos e realiza operações aritméticas básicas (soma, subtração e multiplicação).

Objetivo

Demonstrar o uso correto de entrada de dados e conversão explícita de tipos em Python para garantir cálculos numéricos válidos.

Conceitos praticados

Entrada de dados com input()

Conversão de tipos com float()

Operações aritméticas básicas

Formatação de saída

Observações para recrutadores

Este é um exercício de nível iniciante em Python, focado na construção de uma base sólida. O objetivo é praticar a tipagem correta de dados e evitar erros lógicos comuns ao trabalhar com entrada do usuário, como parte de uma evolução diária voltada para entrevistas.

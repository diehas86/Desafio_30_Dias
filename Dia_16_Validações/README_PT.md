O que este código faz
Este código implementa um sistema de cadastro de usuários com validação de nome, idade, email e senha, garantindo que apenas dados válidos sejam aceitos.

Objetivo
Demonstra como criar funções de validação de entrada e controlar o fluxo de dados do usuário usando loops em Python.

Conceitos praticados

Funções com retorno booleano

Manipulação de strings (strip())

Laços (while) para validação de entrada

Estruturas condicionais (if/else)

Observações para recrutadores
Código desenvolvido por estudante de Python, focado em construir uma base sólida e se preparar para entrevistas técnicas de nível júnior.

What this code does

Implements a simple interactive shopping list system using a terminal menu, allowing users to add items, list all items, remove items by index, and exit the program.

Purpose

Demonstrate basic control flow and data structure handling in Python through a practical, menu-driven exercise.

Concepts practiced

Lists

While loop

Conditional statements

User input handling

Enumerate

Basic validation

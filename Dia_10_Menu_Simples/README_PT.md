O que este código faz

Implementa um sistema simples de lista de compras interativa no terminal, permitindo adicionar itens, listar todos os produtos, remover itens pelo índice e encerrar o programa.

Objetivo

Demonstrar o uso de estruturas básicas de controle e manipulação de dados em Python por meio de um exercício prático com menu.

Conceitos praticados

Listas

Laço while

Estruturas condicionais

Entrada de dados do usuário

Enumerate

Validação básica

Observações para recrutadores

Este é um exercício de nível iniciante em Python, com foco em consolidar fundamentos da linguagem e raciocínio lógico por meio de prática diária.

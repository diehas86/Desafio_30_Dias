O que este código faz
Recebe um texto digitado pelo usuário, contabiliza o total de palavras e quantas dessas palavras possuem mais de cinco caracteres.

Objetivo
Demonstrar o uso de tratamento de texto, validação de entrada e lógica de contagem utilizando estruturas fundamentais do Python.

Conceitos praticados

Entrada de dados do usuário

Limpeza de strings

Listas e iteração

Estruturas condicionais

Uso de contadores

Observações para recrutadores
Exercício de nível iniciante, com foco em lógica clara, organização do código e aplicação correta dos fundamentos, voltado à preparação para entrevistas de Python Júnior.

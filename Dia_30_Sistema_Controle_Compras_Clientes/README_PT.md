O que este código faz
Este programa processa uma lista de registros de compras, calcula o total gasto por cada cliente e identifica qual cliente realizou o maior gasto.

Objetivo
Este exercício demonstra a capacidade de estruturar lógica em funções, agregar dados utilizando dicionários e realizar comparações para extrair informações relevantes de um conjunto de dados.

Conceitos praticados

Definição de funções e retorno de valores

Criação e atualização de dicionários

Iteração sobre listas e dicionários

Estruturas condicionais

Agregação básica de dados

Observações para recrutadores
Este projeto faz parte de uma rotina diária de estudos com foco na construção de uma base sólida em Python para entrevistas de nível Júnior. Neste último dia, consolido conceitos centrais como acumulação de dados, separação de responsabilidades com funções e lógica de comparação.

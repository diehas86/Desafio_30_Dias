What this code does
This program processes a list of purchase records, calculates the total amount spent by each customer, and identifies which customer spent the most.

Purpose
This exercise demonstrates the ability to structure logic into functions, aggregate data using dictionaries, and perform comparisons to extract meaningful results from a dataset.

Concepts practiced

Function definition and return values

Dictionary creation and updates

Iteration over lists and dictionaries

Conditional logic

Basic data aggregation

O que este código faz
Implementa um sistema de verificação de senha permitindo ao usuário um número limitado de tentativas. O programa concede acesso se a senha correta for digitada e bloqueia após todas as tentativas.

Objetivo
Demonstra compreensão de laços, condicionais e controle de estado usando variáveis em Python.

Conceitos praticados

Laço while

Condicionais if/else

Manipulação de variáveis

Entrada e saída básicas

*** Observações para recrutadores
Exercício de nível iniciante, focado em controlar o fluxo do programa sem usar break, try/except, bibliotecas externas ou estruturas avançadas. Mostra o tratamento de um cenário simples do dia a dia com lógica clara.

Regras básicas seguidas

Não foi usado break

Não foi usado try/except

Não foram usadas bibliotecas externas

Não foram usados laços for

Foram usadas apenas variáveis simples

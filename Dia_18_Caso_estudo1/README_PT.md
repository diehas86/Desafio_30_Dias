Este exercício demonstra como estruturar um programa interativo pequeno, validar entradas do usuário sem tratamento avançado de erros e organizar dados usando conceitos básicos de Python, adequados ao nível júnior.

Conceitos praticados

Funções com retorno

Validação de entrada sem try/except

Laços while e controle de fluxo

Listas e dicionários

Estrutura de menu interativo

Observações para recrutadores

Este código faz parte de uma prática diária de Python com foco na construção de uma base sólida para nível júnior. A prioridade é clareza, lógica consistente e capacidade de explicação em entrevistas técnicas.
